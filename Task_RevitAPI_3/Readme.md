# Описание плагина LevelsProperty

## Получение уровней
Получает единицы измерения длины из модели.
Получает из модели коллекцию всех уровней.
Для каждого уровня создается пользовательский класс уровня, для которого заполняются имя, id, высота и высота приведенная к единицам измерения в модели.

## Взаимодействие с пользователем
Открывается окно со списком всех уровней. 
Можно добавлять/удалять/меняя значения параметров имя и высота.

## Результат работы
После нажатия кнопки ОК, изменения применяются в модели.
